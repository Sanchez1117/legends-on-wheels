# Legends on Wheels Website

This project is a two-page prototype for **Legends on Wheels**, a site celebrating classic and muscle cars.  
The design uses a chrome + black + smoky theme with gold accents to give a cinematic feel.

## Project Purpose
The purpose of this project is to create an engaging, visually dynamic prototype that showcases a passion for automotive history, combining design, functionality, and storytelling.

## Features
- Two-page prototype (Home & About)
- Smoky background with chrome style
- Featured car gallery
- Responsive design for desktop and mobile
- Smooth navigation between pages

## Deliverables
- `index.html`  
- `about.html`  
- `style.css`  
- `images/` folder with car images and smoke.jpg  
- `docs/` folder containing:
  - `scope.md`
  - `plan.md`
  - `retrospective.md`
